# 더몬스터학원 블로그 자동화 — 설치 안내

크롬 확장프로그램입니다. 새 컴퓨터에서도 **딱 3단계**면 됩니다.

---

## 처음 설치 (새 컴퓨터)

### 1단계 — 파일 자동 받기 (한 줄만)

키보드에서 **윈도우키 + R** → `cmd` 입력 → Enter → 아래 **한 줄**을 통째로 복사해 붙여넣고 Enter:

```
mkdir C:\blog 2>nul & cd /d C:\blog & curl.exe -L -o install.bat "https://raw.githubusercontent.com/themonsteredu/monster-blog/main/extension/install.bat" & install.bat
```

→ 파일이 자동으로 받아지고, **크롬 확장 설정창**과 **C:\blog 폴더**가 저절로 열립니다.

### 2단계 — 크롬에 확장 올리기

방금 열린 크롬 `chrome://extensions` 화면에서:

1. 오른쪽 위 **개발자 모드** 켜기 (토글)
2. 왼쪽 위 **압축해제된 확장 프로그램을 로드합니다** 클릭
3. 폴더 선택창에서 **C:\blog** 선택
4. (편의) 크롬 우측 위 퍼즐🧩 → 이 확장을 **핀 고정**

### 3단계 — API 키 입력 (이 컴퓨터에서 한 번만)

확장 아이콘 클릭 → **⚙️ 설정** 펼치기:

- **Claude(앤트로픽) API 키** — 글 생성용 (필수)
- **OpenAI API 키** — AI 실사/일러스트 이미지 생성용 (선택, 비우면 카드 이미지)
- **설정 저장** 클릭

끝입니다.

---

## 다음부터 업데이트 (아주 쉬움)

새 기능이 나오면:

1. **C:\blog 폴더** 열기 → **update.bat 더블클릭** ("DONE!" 뜨면 아무 키)
2. `chrome://extensions` → 확장 카드의 **새로고침(🔄)** 클릭
3. 팝업 열어서 버전 번호가 올라갔는지 확인

폴더가 항상 **C:\blog** 로 고정되어 있어서, 크롬에 다시 올릴 필요 없이 새로고침만 하면 됩니다.

---

## 사용 순서

1. **네이버 블로그 글쓰기 화면**을 먼저 엽니다 (blog.naver.com → 글쓰기)
2. 확장 아이콘 클릭
3. 글 종류·주제·키워드·핵심 내용 입력, 필요하면 사진 올리기
4. **발행 방식** 선택 (임시저장만 / 바로 발행 / 예약 발행)
5. **✨ 글 생성** → 내용 확인 (직접 수정 가능)
6. **🚀 네이버에 입력** → 글쓰기 화면 오른쪽 위 초록 상자로 진행 상황 표시

입력 중 화면 위에 "디버깅을 시작했습니다" 띠가 떠도 **정상**입니다(사진·인용구 넣는 동안만). 거기 **'취소'는 누르지 마세요.** 그리고 **F12 개발자도구는 닫고** 실행하세요.

## 자주 묻는 것

- **API 키는 안전한가요?** 이 브라우저(크롬)에만 저장되고 외부로 전송되지 않습니다.
- **다른 컴퓨터에서도 쓰려면?** 위 "처음 설치"를 그 컴퓨터에서 한 번만 반복하면 됩니다. (설정·키는 컴퓨터마다 따로 입력)
- **뭔가 자동으로 안 될 때?** 완료 상자에 뜨는 "진단: …" 줄을 캡처해서 문의하세요. 그걸 보고 바로 고칠 수 있습니다.
