# 설치 안내서 (윈도우 기준) — 클로드코드로 실행하기

> 컴퓨터를 잘 몰라도 따라 할 수 있게 한 단계씩 적었습니다.
> 막히면 그 화면을 캡처해서 클로드코드에게 "이 화면에서 막혔어"라고 물어보세요.

---

## 먼저 알아둘 것

- **블로그 프로그램은 "파이썬"으로 돌아갑니다.** (클로드코드가 아니라)
- **클로드코드는 코드를 대신 실행·수정해 주는 비서입니다.** 한국어로 시키면 됩니다.
- **클로드코드는 유료 구독이 필요합니다.** Claude **Pro 또는 Max** 구독(또는 API 결제 계정).
  무료 claude.ai 계정으로는 클로드코드를 쓸 수 없습니다.

---

## 1단계 · 파이썬 설치 (프로그램 엔진)

1. https://www.python.org/downloads/ 접속
2. 노란색 **"Download Python"** 버튼 클릭해서 설치 파일 내려받기
3. 내려받은 파일 실행
4. ⚠️ **첫 화면 아래쪽 "Add Python to PATH" 체크박스를 꼭 켜세요.** (안 켜면 나중에 명령어가 안 먹힙니다)
5. **"Install Now"** 클릭 → 끝

---

## 2단계 · 클로드코드 설치

1. 시작 버튼 옆 검색창에 `powershell` 입력 → **Windows PowerShell** 실행
2. 검은 창에 아래 한 줄을 붙여넣고 Enter:

   ```powershell
   irm https://claude.ai/install.ps1 | iex
   ```

   (관리자 권한 필요 없음, 이후 자동 업데이트됨)

3. 설치가 끝나면 PowerShell 창을 닫았다가 다시 엽니다.

---

## 3단계 · 로그인

1. PowerShell에 `claude` 입력 후 Enter
2. **브라우저가 자동으로 열립니다** → Claude **Pro/Max 계정**으로 로그인
3. 로그인은 한 번만 하면 됩니다.

---

## 4단계 · 프로그램 받아서 실행

1. 바탕화면에 빈 폴더 하나 만들기 (예: `블로그자동화`)
2. PowerShell에서 그 폴더로 이동:

   ```powershell
   cd $HOME\Desktop\블로그자동화
   ```

3. `claude` 입력해서 클로드코드 실행
4. 클로드코드에게 한국어로 차례대로 시키기:

   > 이 깃허브 저장소를 여기에 내려받아줘: themonsteredu/monster-blog

   다 받아지면:

   > 필요한 패키지 설치하고, streamlit run app.py 로 실행해줘

5. 잠시 뒤 **브라우저에 블로그 자동화 화면이 뜹니다.** 🎉

---

## 5단계 · 처음 사용 설정

화면이 뜨면:

1. **⚙️ 설정** 탭
   - 앤트로픽(Claude) API 키 — 글 생성용
   - OpenAI 키 — 이미지 생성용
   - 네이버 아이디 / 비밀번호 — 임시저장용
   - 썸네일 배경 이미지(.png) 경로, 한글 폰트(.ttf) 경로
   - **저장** 클릭
2. **✍️ 글 만들기** 탭 → 주제·키워드 입력 → **글 생성**
3. **🖼️ 이미지·썸네일** 탭 → 이미지·썸네일 생성
4. **🚀 네이버 올리기** 탭 → 임시저장 → 네이버에서 직접 확인 후 발행

---

## 자주 막히는 곳

- **"python을 찾을 수 없습니다"** → 1단계에서 "Add Python to PATH"를 안 켠 것. 파이썬 재설치하며 체크.
- **네이버 로그인 중 캡차/보안문자** → 그 순간 사람이 직접 창에서 풀어야 합니다.
- **네이버 화면이 바뀌어 로봇이 멈춤** → 클로드코드에게 "네이버 글쓰기 화면이 바뀐 것 같아, naver_robot.py 좀 고쳐줘"라고 요청.

---

## 보안 메모

- API 키·네이버 비밀번호는 `settings.json`(내 컴퓨터)에만 저장되고 외부로 전송되지 않습니다.
- `settings.json`은 `.gitignore`로 제외되어 깃허브에 올라가지 않습니다.
- 학생 등 미성년자 사진은 보호자 동의 없이 공개·업로드하지 않습니다.
