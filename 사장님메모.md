# 사장님 메모 (나만 보는 요약)

> 이 프로그램을 어떻게 설치하고, 어떻게 파는지 잊어버리지 않으려고 적어둔 메모.

---

## 1. 내가 새 컴퓨터에 설치하는 법

1. **윈도우키 + R** → `cmd` → Enter
2. 아래 한 줄 붙여넣고 Enter:
   ```
   mkdir C:\blog 2>nul & cd /d C:\blog & curl.exe -L -o install.bat "https://raw.githubusercontent.com/themonsteredu/monster-blog/main/extension/install.bat" & install.bat
   ```
3. 자동으로 열린 크롬 화면 → **개발자 모드 켜기** → **압축해제된 확장 프로그램을 로드합니다** → **C:\blog** 폴더 선택
4. 프로그램 ⚙️설정 → **Claude API 키** + **우리 학원 정보** 입력 → 저장

## 2. 업데이트 하는 법 (새 기능 나왔을 때)

1. **C:\blog 폴더 → `update.bat` 더블클릭** ("DONE!" 뜨면 아무 키)
2. 크롬 `chrome://extensions` → 프로그램 카드 **새로고침(🔄)**
3. 팝업에서 버전 번호 올라갔는지 확인

## 3. 사용 순서

1. 네이버 블로그 **글쓰기 화면** 먼저 열기
2. 프로그램 아이콘 클릭 → 주제·키워드 입력 (사진 있으면 올리기)
3. **발행 방식** 고르기 (임시저장 / 바로 발행 / 예약 발행)
4. **✨ 글 생성** → 확인 → **🚀 네이버에 입력**
5. "디버깅을 시작했습니다" 떠도 정상. '취소' 누르지 말 것. F12는 닫고 사용.

---

## 4. 남한테 팔 때 (고객 설치)

- 고객에게는 **고객설치안내.md** 내용을 그대로 보내면 됨.
- 고객도 위 1번과 똑같이 설치하고, 자기 **학원 정보 + 자기 API 키**를 넣으면 자기 블로그용이 됨.
  (프로그램 설정의 "🏫 우리 학원 정보" 칸만 바꾸면 글·배너·지도가 그 학원으로 나옴)

### 팔 때 걸림돌 2가지 (꼭 기억)

1. **API 키를 고객마다 발급해야 함** — 비개발자에겐 제일 어려운 부분.
   - 지금 방법: 내가 **원격(애니데스크 등)으로 설치+키 입력까지 해주고 설치비 받기**.
   - 나중 방법: 내 서버로 **키를 대신 제공**해서 고객은 키 없이 쓰게 하고 **월 구독료** 받기. (개발 더 필요)
2. **"개발자 모드로 폴더 올리기"가 낯설다** — 크롬 정책상 지금은 이 방법뿐.
   - 없애려면: **크롬 웹스토어 등록**(1회 등록비 약 $5, 심사) → 고객은 "설치" 버튼 한 번으로 끝.

### 추천 순서

1. 처음 몇 명은 **원격 설치 + 설치비/월 관리비**로 시작해 반응 보기.
2. 잘 되면 → **키 대행 서버**(고객이 키 없이 쓰기) + **웹스토어 등록**(설치 버튼 하나)으로 완성품 만들기.
