# 블로그 자동화 프로그램 설치 안내 (고객용)

안녕하세요! 블로그 자동화 프로그램 설치 방법입니다. 컴퓨터를 잘 몰라도 **3단계**면 끝나요.
(윈도우 + 크롬 브라우저에서 사용합니다.)

---

## 1단계. 파일 자동 받기 (한 줄만 복사)

1. 키보드에서 **윈도우키 + R** 을 동시에 누릅니다.
2. 뜨는 작은 창에 `cmd` 라고 치고 **Enter**.
3. 검은 창이 열리면, 아래 **한 줄**을 통째로 복사해서 붙여넣고 **Enter**:

```
mkdir C:\blog 2>nul & cd /d C:\blog & curl.exe -L -o install.bat "https://raw.githubusercontent.com/themonsteredu/monster-blog/main/extension/install.bat" & install.bat
```

→ 잠시 뒤 **크롬 설정 화면**과 **C:\blog 폴더**가 저절로 열립니다.

## 2단계. 크롬에 프로그램 올리기

방금 열린 크롬 화면(`chrome://extensions`)에서:

1. 오른쪽 위 **개발자 모드** 스위치를 켭니다.
2. 왼쪽 위 **[압축해제된 확장 프로그램을 로드합니다]** 버튼을 누릅니다.
3. 폴더 고르는 창에서 **C:\blog** 를 선택하고 확인.
4. (선택) 크롬 오른쪽 위 퍼즐 모양(🧩)을 눌러 이 프로그램을 **고정**해두면 쓰기 편해요.

## 3단계. 내 정보 입력 (처음 한 번만)

1. 크롬 오른쪽 위에서 프로그램 아이콘을 누릅니다.
2. **⚙️ 설정** 을 펼칩니다.
3. **Claude API 키** 를 붙여넣습니다. (발급 방법은 아래 참고)
4. **🏫 우리 학원 정보** 칸에 학원명·전화·카카오톡·주소·운영시간 등을 채웁니다.
5. **설정 저장** 을 누릅니다.

이제 준비 끝! 네이버 블로그 글쓰기 화면을 열고, 프로그램에서 주제를 넣어 글을 만들면 됩니다.

---

## 사용법 (요약)

1. 네이버 블로그 **글쓰기 화면**을 먼저 엽니다.
2. 프로그램 아이콘 클릭 → 주제·키워드 입력 (사진 있으면 올리기).
3. **✨ 글 생성** → 내용 확인.
4. **🚀 네이버에 입력** → 글이 자동으로 채워집니다. 확인 후 발행!

- 입력 중 화면 위에 "디버깅을 시작했습니다" 띠가 떠도 **정상**이에요. 거기 '취소'는 누르지 마세요.
- 개발자도구(F12)는 닫고 사용하세요.

## 업데이트 (새 기능이 나오면)

1. **C:\blog 폴더**에서 **update.bat** 을 더블클릭 → "DONE!" 뜨면 아무 키.
2. 크롬 `chrome://extensions` 에서 프로그램 카드의 **새로고침(🔄)** 클릭.

---

## Claude API 키 발급 방법 (처음 한 번)

글을 만드는 데 필요한 열쇠입니다. 사용한 만큼만 소액 결제됩니다(글 1개에 수십 원 수준).

1. https://console.anthropic.com 접속 → 가입/로그인
2. 결제수단 등록 후 소액 충전 (예: 5달러)
3. **API Keys** 메뉴 → **Create Key** → 만들어진 키(sk-ant-...)를 복사
4. 프로그램 설정에 붙여넣기

(사진을 AI로 만들고 싶으면 OpenAI 키도 같은 방식으로 발급해 설정에 넣으면 됩니다. 안 넣으면 깔끔한 카드 이미지가 자동으로 들어갑니다.)

---

## 잘 안 될 때

프로그램 화면 오른쪽 위에 뜨는 **"진단: …"** 글자를 그대로 캡처해서 보내주시면 바로 도와드립니다.
