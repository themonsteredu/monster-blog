# 더몬스터학원 블로그 자동화 — 크롬 확장프로그램

파이썬·cmd 없이, **크롬에 확장만 설치**하면 됩니다. 사장님이 평소 쓰는 진짜 크롬 안에서
동작하므로 네이버 보안(캡차 등)에 잘 걸리지 않습니다.

## 설치 (처음 한 번)

1. 이 `extension` 폴더를 컴퓨터에 둡니다. (GitHub에서 ZIP 받아 압축 푼 그 안의 `extension` 폴더)
2. 크롬 주소창에 `chrome://extensions` 입력하고 Enter
3. 오른쪽 위 **개발자 모드** 켜기 (토글)
4. 왼쪽 위 **압축해제된 확장 프로그램을 로드합니다** 클릭
5. 방금 그 **`extension` 폴더**를 선택
6. 크롬 우측 위 퍼즐(🧩) 아이콘 → 이 확장을 **고정(핀)** 해두면 편합니다

## 설정 (처음 한 번)

1. 확장 아이콘 클릭 → 팝업 열기
2. **⚙️ 설정** 펼치기 → **Claude(앤트로픽) API 키** 입력 → **설정 저장**
   - (전용 틀은 비워두면 기본 규칙이 적용됩니다. 학원 정보를 바꾸려면 여기서 수정)

## 사용

1. **네이버 블로그 글쓰기 화면을 먼저 엽니다** (blog.naver.com → 글쓰기)
2. 확장 아이콘 클릭
3. 글 종류·주제·키워드·핵심 내용 입력, 필요하면 **사진 올리기**
4. **✨ 글 생성** → 결과 확인 (제목·본문은 직접 수정 가능)
5. **🚀 네이버에 입력** → 글쓰기 화면에 자동으로 채워짐
6. 내용 확인 후 **발행은 사장님이 직접** 누르기

## 참고 / 한계

- 네이버 에디터 구조는 자주 바뀝니다. 자동 입력이 안 되면 `content.js`의 선택자(SELECTOR)를
  실제 화면에 맞게 수정해야 합니다. (막히면 캡처해서 문의)
- 이미지 자동 삽입은 네이버 화면에 따라 다듬기가 필요할 수 있습니다.
- API 키는 이 브라우저(크롬)에만 저장되며 외부로 전송되지 않습니다.
- 글 생성에는 Claude API 사용량만큼 소액 과금됩니다.
